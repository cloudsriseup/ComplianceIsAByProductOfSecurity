Important info about the files included in the STIG.zip file.


The following files are included in this STIG.  The file names listed below are generic; the actual file names will be specific to the technology and STIG release.


*_Tech_Overview.doc or .pdf – This file will contain the overview and background information that could not be stored in the XML file.


STIG_Manual.zip - These contain the appropriate files.  The STIG_Manual files are for manually viewing the STIG in a browser.  


The STIG_Manaul.zip files need to be extracted to the same directory for use.

STIG Manual.xml – This is the STIG XML file that contains the manual check procedures.

STIG.xsl – This is the transformation file that will allow the XML to be presented in a “human friendly” format.

DoD-DISA-logos-as-JPEG.jpg - Contains logos used by STIG.xsl.
